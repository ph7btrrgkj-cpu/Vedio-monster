# موتيك | Motek - محرك الحركات الذكي بالذكاء الاصطناعي 🎬✨

**موتيك** هو تطبيق متطور لتحرير الفيديو وصناعة الرسوم المتحركة (Motion Graphics) مدعوم بالذكاء الاصطناعي، مصمم خصيصاً للمبدعين العرب. يجمع التطبيق بين جمالية التصميم "الزجاج المصقول" (Frosted Glass) وبين قوة نماذج Gemini AI لتقديم تجربة تحرير سينمائية.

---

## 🚀 المميزات الرئيسية

- **🤖 توليد الحركات بالذكاء الاصطناعي:** اكتب وصفاً للحركة التي تريدها (مثلاً: "دوران شعار مع تلاشي تدريجي") وسيقوم التطبيق بتوليد وتنفيذ الحركة فوراً.
- **🔍 اكتشاف المشاهد ذكياً:** تحليل مقاطع الفيديو المرفوعة تلقائياً لاكتشاف لقطات الانتقال وتوفير وصف باللغة العربية لكل مشهد.
- **🎨 إدارة الخطوط العربية:** نظام متكامل لرفع الخطوط المخصصة واستخدامها في طبقات النصوص مع دعم كامل للغة العربية.
- **💎 تصميم Frosted Glass:** واجهة مستخدم عصرية تعتمد على الشفافية والظلال الناعمة مع دعم كامل للاتجاه من اليمين إلى اليسار (RTL).
- **⏱️ خط زمني احترافي:** تحكم كامل في توقيت الطبقات، مدة الحركات، وخصائص العناصر (المقياس، الدوران، الشفافية).
- **📱 دعم مختلف الأبعاد:** إمكانية التصدير بقياسات Reels (9:16)، YouTube (16:9)، و Square (1:1).

---

## 🛠️ التقنيات المستخدمة

- **React 18** مع **Vite**
- **Tailwind CSS 4** للتصميم
- **Framer Motion** للتحريكات السلسة
- **Gemini AI API (@google/genai)** للذكاء الاصطناعي
- **Lucide React** للأيقونات

---

## ⚙️ البدء والتشغيل

1. **قم بتحميل المشروع:**
   ```bash
   git clone [your-repository-url]
   cd motek-ai
   ```

2. **تثبيت الاعتمادات:**
   ```bash
   npm install
   ```

3. **إعداد مفتاح الـ API:**
   قم بإنشاء ملف `.env` وأضف مفتاح Gemini الخاص بك:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **تشغيل التطبيق:**
   ```bash
   npm run dev
   ```

## 📂 هيكلة المشروع

- `/src/components`: المكونات واجهة المستخدم.
- `/src/lib`: المكتبات (إدارة الخطوط، تكامل Gemini).
- `/src/types`: تعريفات TypeScript للبيانات.
- `/src/constants`: القوالب والحركات الجاهزة.
